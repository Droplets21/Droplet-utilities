# Droplet-utilities
A mod of Glue3D that adds a lot of new features, and allows custom plugins, new categories, and more

Glue3D was made by Gl00b/The3DScratcher, with all rights reserved. With permission to edit the project with credits
Droplet utilities was made by Droplets, with help from users like m3tt, savvy, qloak, nullis, and others

Droplet utils first aimed to connect developers together with live editing features, Droplet utils now aims to replace the default Glue3D user expirience by adding fixes, changes, and new features.
